# Analyze ML Competition

Analyze this machine learning competition/challenge and provide:

1. **Problem Overview**: What are we predicting/optimizing?
2. **Constraints**: Model size, compute budget, time limits, evaluation rules
3. **Data Analysis**: Key characteristics, potential preprocessing needs
4. **Baseline Strategy**: Simplest approach that should work
5. **Technique Research**: What SOTA methods are relevant? Search for recent papers.
6. **Experiment Plan**: Ordered sequence of experiments to run
7. **Risk Assessment**: What could make this fail?

Be specific and quantitative. Cite sources for any technique claims.
